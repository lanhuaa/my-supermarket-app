# 后台数据看板（Streamlit 示例）

这是一个使用 **Python + Streamlit** 构建的简易后台数据看板示例，数据全部为 **模拟数据（Mock Data）**，方便没有后端服务的情况下快速演示。

## 功能模块

- **概览**：总销售额、用户数、订单数（卡片形式展示）
- **销售趋势**：最近 7 天销售额折线图
- **用户分布**：按地区展示的用户占比饼图
- **最新订单**：最近订单表格列表

## 环境准备

1. 安装 Python（建议 3.9+）
2. 在项目根目录执行：

```bash
pip install -r requirements.txt
```

## 运行方式

在项目根目录下执行：

```bash
streamlit run app.py
```

执行后浏览器会自动打开本地页面（如果没有自动打开，可以把命令行中显示的本地地址复制到浏览器访问）。

---

本项目仅用于演示 Streamlit 仪表盘界面结构和交互效果，所有数据均为随机生成。
